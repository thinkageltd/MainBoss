Quitting MainBoss

< Previous section  |  Table of Contents  |  Index  |  Next section >

To quit MainBoss, simply close all open MainBoss windows (by clicking the X in the upper right hand corner). Alternatively, you can select Session →> Exit from the menu.

Exiting the control-panel window does not close any other MainBoss windows you have open. They must each be closed individually.
