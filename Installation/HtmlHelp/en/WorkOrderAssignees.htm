Work Order Assignees

< Previous section  |  Table of Contents  |  Index  |  Next section >

The work order assignee list specifies people who can be assigned to work orders. Assigning someone to a work order means that person has some special interest in the work order; for more, see Assignments.

Any worker who appears in a labor demand (inside or outside, hourly or per job) is automatically assumed to be an assignee for the associated work order, provided that the worker is on the Work Order Assignee list. The worker's name will therefore appear in the Work Order Assignments section of the work order.

For information on viewing work order assignee records, see Viewing Work Order Assignee Records. For how to create and edit work order assignee records, see Editing Work Order Assignee Records. For printing work order assignee records, see Printing Work Order Assignee Records.
