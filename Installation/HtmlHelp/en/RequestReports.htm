Request Reports

< Previous section  |  Table of Contents  |  Index  |  Next section >

The Requests | Reports section of the control panel lets you obtain the following reports about your requests:

  • Requests by Assignee Report: prints off requests in batches, one batch for each assignee; for example, if you have workers A, B and C assigned to requests, the print-out consists of all A's assigned requests, followed by all B's assigned requests, then all C's assigned requests
Request Summary Report: a summary of information on requests
  • Request History Report: information about past requests
Request Summary by Assignee and Request History by Assignee Reports: information on past requests, grouped by assignee
  • Request State History Report: a history of when requests changed their state or status
Request State History Summary Report: a summary of when requests changed their state or status
