Void Codes

< Previous section  |  Table of Contents  |  Index  |  Next section >

If you make a mistake when recording a physical count, MainBoss lets you void (cancel) the physical count record. In this situation, you are asked to specify a void code. This code explains why you decided to void the physical count. Here are some sample void codes:

Mistake in counting
Mistake in number entry

For information on viewing void codes, see Viewing Void Codes. For information on creating and editing void codes, see Editing Void Codes. For information on printing void codes, see Printing Void Codes.
