Editing Task Specializations

< Previous section  |  Table of Contents  |  Index  |  Next section >

A task specialization is a specialized version of another task. (For more, see Task Specializations.)

To create a task specialization, you use New Task Specialization from Unit Maintenance Plans | Tasks or from Coding Definitions | Unit Maintenance Plans | Tasks.

The contents of a specialized task are similar to a normal task. Therefore, editing a specialized task is the same as editing a normal task, with all the information fields having the same meaning. For more on editing tasks, see Editing Tasks.
