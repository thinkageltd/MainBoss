The Database Server List

< Previous section  |  Table of Contents  |  Index  |  Next section >

If you click the "..." associated with "Database Server" in windows for specifying databases, MainBoss displays a list of SQL Servers that are known to your Windows system. This may not be a complete list of the servers you can actually use; in particular, it's possible for a system to set up its firewall in such a way that it has a SQL Server available for use but other computers aren't told about the server unless they ask. In this situation, your Windows system won't know the server is available, but you can use the server if you explicitly specify its name.

For more on SQL Server, including a number of configuration tips, see the Installation and Administration Guide.
