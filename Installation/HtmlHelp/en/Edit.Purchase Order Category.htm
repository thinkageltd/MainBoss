Editing Purchase Order Categories

< Previous section  |  Table of Contents  |  Index  |  Next section >

You create or modify purchase order categories using the purchase order category editor. The usual way to open the editor is to click New Purchase Order Category or Edit in the View section of Coding Definitions | Purchase Orders | Purchase Order Categories.

The purchase order category editor window contains the following:

Details section: Shows basic information for the record.

Code: A brief code to identify this record. No two records may have the same code.

Description: A longer description of the purchase order category.

Comments: Any comments you want to associate with the purchase order category.

Purchase Orders section: Shows any purchase orders that belong to the selected purchase order category. Before you can record any purchase orders in this list, you must Save the purchase order category record. For more on purchase orders, see Purchase Orders.

Purchase Order Templates section: Shows any purchase order templates associated that belong to the selected purchase order category. Before you can record any templates in this list, you must Save the purchase order category record. For more on purchase order templates, see Purchase Order Templates.

Save: Saves the current record. The editor window stays open in case you want to make more changes to the same record.

Save & New: Saves the current record and sets up the window for you to enter a new record. Fields in the new record will be blank or set to default values.

Save & Close: Saves the current record and closes the editor window.

Cancel: Closes the window without saving any changes you've made to the record since the last time you saved. When you Save the record, the Cancel button changes to Close, indicating that you can safely close the window without losing any changes.

Close: Closes the window. This button only appears after you've saved changes with Save or before you've entered any data at all. Otherwise, the button is labeled Cancel.

For more on purchase order categories, see Purchase Order Categories. For more on viewing purchase order categories, see Viewing Purchase Order Categories. For more on editors in general, see Using Editors.
