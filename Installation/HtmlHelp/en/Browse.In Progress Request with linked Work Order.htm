Viewing In Progress Requests with Work Orders

< Previous section  |  Table of Contents  |  Index  |  Next section >

The control panel entry Requests | In Progress Requests | In Progress Requests with linked Work Order lists any requests that are "In Progress" and have at least one work order linked to them. Requests in this table have been processed by the maintenance department, and typically need no further attention until the associated work orders are closed.

The fields of this table viewer are exactly like the fields of Requests | In Progress Requests. For more information, see Viewing In Progress Requests.
