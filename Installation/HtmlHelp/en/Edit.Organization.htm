Specifying Maintenance Organizations

< Previous section  |  Table of Contents  |  Index  |  Next section >

This window can be used for the following purposes:

  • To add a new database to your list of available databases. For more information, see Existing Maintenance Organizations.
  • To create a new database. For more information, see New Maintenance Organizations.
  • To create a new database using data obtained from a backup file. For more information on this operation, see Creating a New Maintenance Organization from a Backup File.
  • To create a new database using data imported from a MainBoss Basic database. For more information, see Imported Maintenance Organizations.
  • To create a new database with demonstration mode license keys. This database will let you test MainBoss operations, but puts limitations on the database contents (e.g. a limited number of work orders, locations, requests, etc.). For more information, see Demonstration Maintenance Organizations.
  • : Lets you change information about the organization (e.g. if you want to change the "Organization Name" which is displayed when you are using MainBoss). For more on this operation, see Changing Organization Information.
