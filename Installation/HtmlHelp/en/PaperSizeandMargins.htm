Paper Size and Margins

< Previous section  |  Table of Contents  |  Index  |  Next section >

When you start MainBoss, it sets the margin sizes for any report to default values, and sets the paper size to the default size associated with your default printer. You can change both the margins and the paper size by clicking the Page Setup button in the Preview section of any window that prints reports.

Note: Any change in margin size or paper size only affects the current printing window. Different reports will still have the default margin size and paper size. Also, if you close a report window then open it again, the margin size and paper size for that report will be set back to the defaults.

When you export data in PDF format, MainBoss uses the margin sizes and paper size associated with your current Page Setup. This means that you might use Page Setup to specify a paper size and/or margin sizes that your printer can't actually handle, if you intend to export the report in PDF format.
