External Tags

< Previous section  |  Table of Contents  |  Index  |  Next section >

An external tag is a value that can be represented as a bar code. (For more on bar codes, see Bar Code Support.) This includes the numbers of all requests, work orders and purchase orders, as well as values specified in the "Unit Tag" fields of unit records and the "External Tag" fields of storeroom and storeroom assignment records.

For information on viewing external tags, see Viewing Your External Tags.
