The About Screen

< Previous section  |  Table of Contents  |  Index  |  Next section >

The About window displays information about MainBoss. To see the window, select Help →> About from the menu. The window includes the following:

  • What version of MainBoss you're using
  • What database you're using and the SQL Server that allows you access to the database
  • Copyright information
  • An Ok button; click this to close the window
