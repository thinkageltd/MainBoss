The Actions Menu

< Previous section  |  Table of Contents  |  Index  |  Next section >

The contents of the Actions menu depend on what you've currently selected in the control panel. For example, if you've currently selected Units in the control panel, the Actions menu will have entries for creating a new unit, editing an existing one, deleting a unit, and so on. Actions entries usually correspond to buttons in the same window (e.g. New Unit corresponds to the New Unit button and has the same effect).
