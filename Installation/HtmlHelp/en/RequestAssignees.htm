Request Assignees

< Previous section  |  Table of Contents  |  Index  |  Next section >

The request assignee list specifies people who can be assigned to requests. Assigning someone to a request means that person has some special interest in the request; for more, see Assignments.

For information on viewing request assignee records, see Viewing Request Assignee Records. For how to create and edit request assignee records, see Editing Request Assignee Records. For printing request assignee records, see Printing Request Assignee Records.
