Configuration de l'organisation

Vous êtes dans le programme de configuration de l'accès à la base de données de l'organisation. Cela vous permettra de

  • Ajouter ajouter un accès à une base de données MailBox existante
  • Créer une nouvelle base de données de démonstration sur un serveur SQL
  • Créer une nouvelle base de données vide avec un accès limité sur un serveur SQL
  • Restaurer une base de données MailBox existante à partir d'un fichier de sauvegarde
  • Créer une nouvelle base de données à partir de MainBoss Basic exporté des fichiers de données
  • Mise à niveau d'une base de données MainBoss vers un nouveau niveau de version
  • Autres activités de maintenance de la base de données

Aide à Démarrer

Si vous êtes nouveau sur MainBoss Advanced, nous avons des instructions de démarrage rapide sur notre site Web au Démarrage Rapide.
