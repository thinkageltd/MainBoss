Configuración de la Organización

Está en el programa de configuración de acceso a la base de datos de la organización. Esto le permitirá

  • Agregar agregar acceso a una base de datos de buzones existente
  • Crear una nueva base de datos de demostración en un servidor SQL
  • Crear una nueva base de datos vacía con acceso limitado en un servidor SQL
  • Restaurar una base de datos de buzón de correo existente desde un archivo de copia de seguridad
  • Crear una nueva base de datos de MainBoss Basic exportó archivos de datos
  • Actualización de una base de datos MainBoss a un nuevo nivel de versión
  • Otras actividades de mantenimiento de bases de datos
