Organization Setup

You are in the organization database access configuration program. This will allow you to

  • Add add access to an existing MainBoss database
  • Create a new demonstration database on a SQL server
  • Create a new empty database with limited access on a SQL server
  • Restore an existing MainBoss database from a backup file
  • Create a new database from MainBoss Basic exported data files
  • Upgrading a MainBoss database to a new version level
  • Other database maintenance activities
